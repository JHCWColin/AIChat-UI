## **标题：** v1.0.0 ｜ 首个公开版本发布：AI聊天客户端，支持DeepSeek/GPT/Gemini等多模型

**更新内容：**
很高兴发布 **AI Chat UI Demo** 的第一个正式版本！这是一个基于Web的AI聊天界面，支持多模型切换、本地数据存储与云端同步。

**✨ 主要特性**
- **多模型支持**：集成 DeepSeek、GPT-5、Gemini-3、Grok 等多个系列模型，可在输入框上方快速切换。
- **精美界面**：采用“液态玻璃”毛玻璃设计，支持**浅色/深色主题**一键切换。
- **对话管理**：
    - 支持创建多个独立对话，并实时搜索历史对话。
    - 对话标题自动根据首条消息生成。
- **数据安全**：
    - 本地数据保存在浏览器 `localStorage` 中。
    - 支持**手动同步**至LeanCloud云端，防止数据丢失（需自行配置或使用内置Key）。
- **用户体验**：
    - 支持**流式输出**（打字机效果）。
    - 内置**代码高亮**与**一键复制**功能。
    - 可调节**回复温度（严谨/适中/创造）** 和**字体大小**。
    - 支持**中止生成**与**单条消息重试**。
- **跨平台**：手机端适配良好，侧边栏可自动隐藏。

**🛠️ 技术栈**
- HTML5 / TailwindCSS / 原生JavaScript
- Marked.js / Highlight.js
- LeanCloud 数据存储
